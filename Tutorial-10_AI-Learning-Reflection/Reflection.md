### 1. Learning Experience Summary

**Before AI - My Old Way:**
1. Open 10 Google tabs → Get confused
2. Watch 1hr YouTube → Still doubts remain  
3. Stuck on Git error → Waste 2 hours
4. Write notes → Copy-paste, no understanding

**Result**: Tasks 1-9 completed in 10 days vs earlier 1 month. AI became my "24x7 personal tutor".

### 3. Challenges I Faced

| Challenge | What Happened | How I Solved It |
| --- | --- | --- |
| **Over-dependence** | Day 3: Tried to copy AI answer directly for Task 4 | Sir said "add your own words". Now I always rewrite AI output |
| **Accuracy issues/Hallucination** | Day 5: ChatGPT gave wrong Git command | Verified on Stack Overflow before running. Now I cross-check medical/facts |
| **Prompt confusion** | Day 1: Asked "tell about AI" → got 5 pages essay | Learned "Role+Context+Format" prompting in Task 7. Much better results now |
| **No internet in free tools** | Claude couldn't give 2026 data | Use Gemini/Perplexity for current info, Claude for PDFs |
| **Too many options** | Confused: ChatGPT or Gemini? | Made Task 5 comparison table. Now I pick tool based on task |

**Key Learning**: AI is 90% right, 10% wrong. Human brain must verify that 10%.

### 4. AI Prompts That Worked Best for Me
These 3 prompt formulas gave me 10x better results:

1. **Teacher Prompt**: `Act as my college teacher. Explain [topic] in Telugu+English for B.Tech 2nd year student. Use examples + table.`
2. **Debug Prompt**: `I got this error: [paste error]. I'm using VS Code + Git on Windows. Give exact 2 commands to fix.`
3. **Summary Prompt**: `Summarize my notes on [topic] into 5 bullet points for exam revision. Use simple words.`

### 5. Future Use Plan - Responsible AI Usage

I will follow this "AI Ethics Code" from now:

1. **AI for Learning, Not Cheating**: Use AI to understand, then close AI and write in my own words. Never direct copy-paste.
2. **Verify Critical Info**: For medical, legal, coding - always check 1 source before trusting AI.
3. **Skill Building**: Use AI to learn "prompt engineering" and "critical thinking", not just answers.
4. **Productivity Use**: Task 7 planner + Task 9 knowledge base will be maintained weekly using AI.
5. **Help Others**: Teach my friends in Nuzvid how to use AI safely for studies.

**My 2026 Goal**: Become "AI-enabled student" not "AI-dependent student".
