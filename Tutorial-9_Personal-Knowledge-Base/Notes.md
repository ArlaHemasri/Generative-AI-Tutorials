# Generative AI - Master Note

## Definition
**Generative AI** = Artificial Intelligence that creates new content: text, images, audio, code, synthetic data.
Unlike old AI that only classifies, GenAI "generates".

**Key Formula**: `Large Dataset + Neural Networks + Prompts = New Content`

## Core Concepts

### 1. How It Works [[Neural Networks]]
- **Base**: Deep neural networks called Transformers
- **Training**: Learns patterns from billions of text/images
- **Generation**: Predicts next word/pixel based on prompt
- **Example**: ChatGPT predicts next word. DALL·E predicts next pixel.

### 2. Requirements [[Datasets]]
- **Data**: Needs huge datasets - GPT-4 trained on ~13 trillion tokens
- **Compute**: Needs GPUs. Training costs $100M+
- **Fine-tuning**: Can adapt base model to specific task with less data

### 3. Types of GenAI [[Model Types]]
| Type | Creates | Example Tool | Use Case |
| --- | --- | --- | --- |
| **LLM** | Text | ChatGPT, Claude, Gemini | Writing, coding, chat |
| **Image Gen** | Images | DALL·E, Midjourney | Art, design, medical scans |
| **Code Gen** | Code | GitHub Copilot, Codeium | Programming help |
| **Audio Gen** | Voice/Music | ElevenLabs, Suno | Voiceover, songs |
